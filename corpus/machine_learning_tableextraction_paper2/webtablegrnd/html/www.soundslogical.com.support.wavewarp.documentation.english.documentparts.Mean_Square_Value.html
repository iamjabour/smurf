Dynamic Range Controllers:

Mean-Square Value



Functional Description
Audio mean-square level measurement. The output control signal is equal to the mean-square amplitude of the input audio signal, with built-in time-averaging (adjustable via the Parameter Window).
Algorithm
See [Zo] section 7.2.1 for information on audio level measurement algorithms. The time-averaging is performed with first-order low-pass filtering.
Signal Implementations
Audio signals Control signals Description
Single input mono single output Control output equals mean-square value of mono audio input, with time-averaging
Single input stereo single output Control output equals mean-square value of the average of the left and right audio input channels, with time-averaging
