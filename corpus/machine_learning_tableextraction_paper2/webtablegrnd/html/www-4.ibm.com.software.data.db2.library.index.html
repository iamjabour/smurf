With the availability of DB2 Universal Database Version 6.1, the Library has been updated with latest versions of the publications. Available on the publications page.

The DB2 Product and Service Technical Library is a searchable database of technical information on DB2 and companion products. There are over 6000 documents in the Library including product publications, redbooks, readme files, and technical notes on bugs, fixes, and frequently asked questions.

The Library has been organized according to the following products:

  • DB2 Universal Database

    Version 6, Version 5 and companion products running on PC and UNIX platforms, including OS/2, Windows 95, Windows NT, Windows 3.1, Linux, AIX, HP-UX, and Solaris.

  • DB2 Server for VSE & VM (SQL/DS)

    Version 5 and Version 3 Release 5 and companion products running on VSE and VM mainframe environments.

  • DB2 Server for OS/390

    Version 5 running in the enterprise environment.

  • DB2 Universal Database for iSeries

    iSeries Version 4 Release 5 online information for Database.

  • DB2 common server

    Version 2 and companion products running on PC and UNIX platforms, including OS/2, Windows 95, Windows NT, Macintosh, Windows 3.1, AIX, HP-UX, Solaris, SINIX, and SCO OpenServer.
