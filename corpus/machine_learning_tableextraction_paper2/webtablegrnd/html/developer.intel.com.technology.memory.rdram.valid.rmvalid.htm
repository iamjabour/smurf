Listed below are the results of testing a small number of RIMM modules on the Intel customer reference motherboard(s). The reference motherboard is intended for evaluation purposes only. We are providing this information to complement RIMM data sheet validation, and as a guide to module performance with the Intel® 820 chipset customer reference platform or Intel® 850 chipset platform (Intel® Desktop Board D850GB) or BOTH platforms. The customer reference motherboard is intended for evaluation purposes only. This testing is not intended to replace the normal OEM module qualification process. For results on specific Intel or OEM production motherboards, refer to the OEM's list of qualified memory suppliers.
