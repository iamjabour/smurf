CME -- London Clearing House Cross-Margin Program

Record Layout for "Cross-Margining Results" File

updated April 03, 2000

The following table provides the record layout for the Cross-Margin Results file which will be made available on request by the CME to clearing members participating in the CME-LCH Cross-Margining program.

The file will be made available by CME at the end of each night's end-of-day clearing cycle. Firms may also elect to receive a file produced after the CME's intraday clearing cycle.

Assuming that the clearing member has exactly one performance bond account participating in the program (as will typically be the case at program inception), the file will contain exactly two records. For the file produced at the CME's end-of-day cycle:

  • One record will contain the results of that day's LCH end-of-day cycle. The delta available for spreading on this record will be used by CME for that day's end-of-day margin cycle and for the next day's intraday cycle. The transmitting clearing organization on this record will be LCH .



  • The other record will contain the results of that day's CME end-of-day cycle. The transmitting clearing organization will be CME.

For the file produced at the CME's intraday cycle:

  • One will contain the results of the previous day's LCH end-of-day cycle. As before, the transmitting clearing organization on this record will be LCH .



  • The other record will contain the results of that day's CME intraday cycle. The delta available for spreading on this record will be used by LCH for that day's end-of-day margin cycle. As before, the transmitting clearing organization on this record will be CME .
