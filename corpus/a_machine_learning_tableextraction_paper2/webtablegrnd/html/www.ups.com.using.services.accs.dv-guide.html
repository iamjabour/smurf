A package is automatically protected against loss or damage up to $100 (U.S.).

Additional protection up to $50,000 (U.S.) per package is available upon request. The charge for each $100 of Insured Value, or fraction thereof, is $.35 (U.S.). The maximum value or Insured Value per package shipped at a UPS Letter Center is $500. For additional limitations, refer to the terms and conditions contained in the UPS Tariff .

To insure a package having a value greater than $100 (U.S.), show the full value in the Insured Value field on the Air Shipping Document, UPS Waybill or UPS shipping system.
