Results Manager Window

The Results window gives you an overview of the applications you started (lower left frame), their status (status column) and their output files (lower right frame). From here you can view and print your results. You can save output files under different names, or delete them. You can also kill a running job in this window.

A log file is displayed for each job. This helps you keeping track if different jobs with the same application were started.

The Refresh button updates the contents of your Results window. Use the "Remove" function to clean up this window.
