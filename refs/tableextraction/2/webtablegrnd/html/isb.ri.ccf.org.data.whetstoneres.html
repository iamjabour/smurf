Please report results for your favourite computer and programming language to Ton van den Bogert <bogert@bme.ri.ccf.org>. If this is a problem, make sure that the benchmark is the only process running, or replace the walltime() system call by another method to measure execution time. In UNIX, for example, the 'time' command may be entered from the shell prompt to measure true CPU time. See source code for additional information.
