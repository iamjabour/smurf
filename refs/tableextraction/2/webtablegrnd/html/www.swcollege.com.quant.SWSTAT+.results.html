Quick Results Using South-Western Stat+
The following exercises illustrate how easy it is to use South-Western Stat+ for statistical data analysis. Print a copy of this web page, and try them.

Enjoy statistics in 22 minutes!

To get up and going quickly in South-Western Stat+ complete the following exercises: from loading the software to printing some output. The most time consuming step will be opening the CD package!

Step 1: Installing South-Western Stat+ (allow about 4 minutes)

  • Check that you have Excel 5 or later on your PC, and you are using Windows 95 or later.
  • Remove the CD from the Package.
  • Place the CD itself into your CD-ROM drive. Wait for the CD to self-load: it may take 30 seconds or so.
  • If the CD does not self-load, use Run, Browse, to run Setup from the CD; or click on the 'setup.exe' file on the CD.
  • Click the Install South-Western Stat+ Add-in button, then follow the instructions on the screen to complete the installation.



Step 2: Running South-Western Stat+ and opening in Excel (about 4 minutes)

  • Click the Start button on the Windows desktop, then from the Programs group select South-Western Stat+ > South-Western Stat+.
  • This should automatically start Microsoft® Excel, and automatically load South-Western Stat+.
  • If the message appears regarding macros, click the button "Enable macros", otherwise South-Western Stat+ will not run.
  • Wait a minute or so for South-Western Stat+ to be loaded in: it takes some time as there are many macros to be loaded.
  • The South-Western Stat+ banner will appear. Click the Continue button to complete the load.
  • You will see that you are still entirely in Microsoft® Excel.
  • However, note the South-Western Stat+ menu item added before Window in the Excel menu: click on the South-Western Stat+ menu item to see the options, and run down to Statistics to see those options. (Don't select any option yet.)
  • Note the South-Western Stat+ toolbar at the right of the screen: Place the mouse pointer over the first button (but do not click) to read its function; try some of the other buttons, but don't select any yet.
  • In summary, you have access to all the Excel options, plus the South-Western Stat+ menu and toolbar options.



Step 3: Data entry and setting South-Western Stat+ workbook format (about 2 minutes)

  • If you wish to use some South-Western Stat+ options, you can type data into your Excel spreadsheet as you normally would. But in this demonstration, we will use a data set we prepared earlier (one that comes on the CD).
  • To obtain this data set, click on File > Open and then go to C:\South-Western Stat+. Open the file Samples2.
  • This file contains a number of data sets that are used in the South-Western Stat+ tours/tutorials and Help system.
  • At the bottom of the Excel worksheet, click on the sheet tab name "DataSet1".
  • The data that appear relate to 20 cars in a used car yard, and there are 4 "numerical" variables for each car.
  • Under Step 4, this data set will be set as a Data Area, but first we set up the workbook for South-Western Stat+ use.
  • To do this click the SWStat+ button on the toolbar or (select menu, South-Western Stat+ > Make South-Western Stat+ Workbook.)
  • The main thing to note is that a new sheet called the Results Sheet is added to the workbook (see tab at bottom). To go to this sheet click on the RS button on the South-Western Stat+ toolbar. (The sheet should be empty.)
  • Return to the car data by clicking on the "DataSet1" tab at the bottom of the screen.



Step 4: Setting a Data Area to use with South-Western Stat+ (about 2 minutes)

  • In order to use most South-Western Stat+ options, you must first set the data you wish to use as a Data Area. Detailed instructions are available in Tour 4 in the User Guide, but the following is a quick introduction.
  • Starting in the cell containing "Car No.", A3, select/highlight the first five columns (up to Style No.) and the rows to the 20th car. That is, you should now have the cells A3 to E23 selected and highlighted.
  • Set this selection as a Data Area with both Column and Row Headers by clicking on the first button on the South-Western Stat+ toolbar. The left column (Car No. 1, etc) and the top row (Seating, Owners, etc) should now be in yellow as Row and Column Headers, and the numerical cells should also be surrounded by a blue border.
  • (You can also use the menu: click on South-Western Stat+ > Data Area > Set Data Area, but close that dialog for now.)
  • Column headers are important in South-Western Stat+ as they identify the variables you wish to use.
  • We are now ready to perform some calculations and create some graphs.



Step 5: Calculating key summary measures with South-Western Stat+ (about 3 minutes)

  • As your Data Area is still set, from the Excel menu, choose South-Western Stat+ > Statistics > Ungrouped Data.
  • Calculating options are given in the left of the dialog; "Sample Data" is set as the default.
  • Note the four variables: "Seating", "Owners", "Price$" and "Style No." are given under "Columns".
  • It does not make sense to calculate summary measures for "Style No.", so uncheck it.
  • Now click the Calculate button. All calculations are written to the Results Sheet.
  • Click the Ops button on the South-Western Stat+ toolbar to go back to the Ungrouped Data dialog.
  • Click Percentile, and type 15 in the edit box.
  • Click the Calculate button to calculate the 15th percentile for all three variables.
  • Note they are added to the bottom of the Results Sheet.
  • Click the Ops button to return to the Ungrouped Data dialog, then click Close to close the dialog.



Step 6: Producing a regression model with South-Western Stat+ (about 3 minutes)

  • Is there a relationship between "Price$" and "Owners" for these used cars?
  • Your car yard data set is still set as the Data Area, and the Results Sheet is still there.
  • >From the Excel menu, choose South-Western Stat+ > Statistics > Regression and Correlation.
  • Calculating options are given in the left of the dialog; "Regression" is set as the default.
  • Note the four variables: "Seating", "Owners", "Price$" and "Style No." are given under "Variables".
  • Click on "Price$" as the Y (dependent) variable and deselect the variables "Seating" and "Style No.".
  • Now click the Calculate button. All calculations are again written to the Results Sheet.
  • If some of the results are shown as #####, as you are in Excel, drag columns D and E to widen them.
  • Click on the Scroll Bar to scroll up the Results Sheet: your earlier calculations are still there.
  • Click the Ops button on the South-Western Stat+ toolbar to go back to the Regression and Correlation dialog.
  • In the bottom left of the dialog, the Fitted option for Plots should be selected: click "Owners" in the list.
  • Click the Show button; answer Yes in the next dialog.
  • The plot is written to the bottom of the Results Sheet. It can be edited using Excel's editing options.
  • Click the Ops button to return to the dialog, then click the Save button; close the dialog.



Step 7: Special graphs with South-Western Stat+ (about 3 minutes)

  • South-Western Stat+ has some speciality graphs. Your Data Area is still set, and the Results Sheet is still there.
  • Click the Graphs button on the toolbar (the 6th from the top) (or choose South-Western Stat+ > Graphs and Charts).
  • Click Box Plot from the Graph Type list. Under variables deselect "Price$". (The box plot for the remaining variables will not make sense, but we draw it here for illustration purposes only.)
  • Click the Show Graph button (then Yes in the next box) and wait while the box plot is created.
  • Scroll up the Results Sheet and see that your previous regression graph is still there, plus other results.
  • Click the Ops button on the South-Western Stat+ toolbar to go back to the Graphs dialog.
  • Under Enhance, click on "1.5 IQR", then Show Graph (and "No") to display fences and outliers.
  • Click the Ops button to return to the dialog, then click the Save Graph button; close the dialog.



Step 8: Printing results and saving your work (about 1 minute)

  • As you are in Excel, use its options to print. From the Results Sheet, click File > Print > OK.
  • If you wish to save your results, chose File > Save, and select a folder and file name (eg SWStat+Intro).
  • Close down Excel if you wish, or spend another five minutes below to explore South-Western Stat+ a little further.



Exploring further if you wish (about 5 minutes), then start at Tour 1!

  • South-Western Stat+ has an extensive help system: In Excel select Help > South-Western Stat+ Help > Main South-Western Stat+ Help.
  • For a brief look select: Statistics and Statistics Routines> Probability Distributions > Normal Distribution.
  • Scroll down through this topic. Click on Contents to explore more, or click on File > Exit to return to Excel.
  • If you have a hardcopy of the South-Western Stat+ User Guide page through to Section C to see the 30+ tours available.
  • If not, open Microsoft® Word, then choose File > Open, and in C:\South-Western Stat+ open file Userguid.doc.
  • Page through the guide, or print it out. The User Guide is also on the CD, and available for students to print.
