Reference Terms: Daily Values and Percent Daily Value



Daily Values and Percent Daily Value (% Daily Value or % DV) are reference terms on the nutrition label.

Daily Values are set by the government and reflect current nutrition recommendations for a 2,000 calorie reference diet. The % Daily Value can be used to compare food products and see how the amount of a nutrient in a serving of food fits into a 2,000 calorie reference diet.

Daily Values: An Umbrella Term

The terms Daily Value and % Daily Value appear on the nutrition label. Technically, the term Daily Value is an "umbrella" term for two sets of reference values: Daily Reference Values (DRVs) and Reference Daily Intakes (RDIs).

Daily Reference Values

Daily Reference Values (DRVs) are set for total fat, saturated fat, cholesterol, total carbohydrate, dietary fiber, sodium, potassium, and protein. DRVs are established by the government for adults and children aged 4 or older, and they are based on current nutrition recommendations. The term Daily Reference Values does not appear on the nutrition label.

DRVs for cholesterol, sodium, and potassium are set at a constant level for all calorie levels. They are based on the National Research Council's Diet and Health Report (1989). The chart below shows DRVs (Daily Values) for these three nutrients.

Daily Reference Values
Food Component Daily Reference Value (Daily Value)
Cholesterol 300 mg
Sodium 2,400 mg
Potassium* 3,500
*Optional on nutrition label.

DRVs for total fat, saturated fat, total carbohydrate, dietary fiber, and protein are based on a 2,000 calorie reference diet. They are based on the following nutrition recommendations.

Nutrition Recommendations
Food Component Calculated as *
Total Fat 30 percent of total calorie intake
Saturated Fat 10 percent of total calorie intake
Total Carbohydrate 60 percent of total calorie intake
Dietary Fiber 11.5 grams per 1,000 calories
Protein 10 percent of total calorie intake
*Some numbers may be rounded for nutrition labeling.

Reference Daily Intakes

Reference Daily Intakes (RDIs) are established for 19 vitamins and minerals. The RDIs are the same as the current U.S. RDAs. The term Reference Daily Intake does not appear on the nutrition label. Click Here to see the chart that lists RDIs for adults and children aged 4 or older.

Note : FDA has proposed new RDIs for seven additional nutrients: vitamin K, selenium, chloride, manganese, fluoride, chromium, and molybdenum.

Percent Daily Value (% Daily Value of % DV)

The % Daily Value shows how the amount of a nutrient in a serving of food fits into a 2,000 calorie reference diet. Consumers can use % Daily Value to compare food products and determine what a serving of food contains relative to a 2,000 calorie reference diet. Consumers can add the % Daily Value for an individual nutrient, such as fat, in all the foods eaten in one day to get an idea of how the day's diet fits within nutrition recommendations for fat. Adding the % Daily Value for a nutrient in all the foods eaten in one day can be an extra tool for educators and consumers who are comfortable with the concept of percentages.

Understanding % Daily Value: Other Calorie Levels

Most people probably do not eat a 2,000 calorie reference diet. However, the % Daily Value information on the nutrition label can still be used to see if a diet meets nutrition recommendations for a nutrient without going through the extra steps to calculate personalized nutrient amounts (Click Here) .

REMEMBER:

  • If you eat less than 2,000 calories, your total % Daily Value for an individual nutrient in all the foods you eat in one day can add up to less than 100 percent.
  • If you eat more than 2,000 calories, your total % Daily Value for an individual nutrient in all the foods you eat in one day can add up to more than 100 percent.

Total fat, saturated fat, total carbohydrate, dietary fiber, and protein levels are based on calorie intake. This means that the total % Daily Value can be adjusted for a specific calorie level. The examples below explain how to add % Daily Value for an individual nutrient for two different calorie levels.

Example 1 . Assume a person eats 1,600 calories a day. The adjusted % Daily Value for 1,600 calories is 80 percent . That means the total % Daily Value for each individual nutrient can add up to 80 percent (instead of 100 percent) for 1,600 calories.

If the total % Daily Value for fat in all the foods he or she eats in one day add up to 80 percent or less, the diet fits within the nutrition recommendations for fat.

Example 2 . Assume a person eats 2,800 calories a day. The adjusted % Daily Value for 2,800 calories is 140 percent . That means the total % Daily Value for each individual nutrient can add up to 140 percent (instead of 100 percent) for 2,800 calories.

If the total % Daily Value for dietary fiber in all the foods he or she eats in one day adds up to 140 percent, the diet falls within the nutrition recommendations for dietary fiber.
