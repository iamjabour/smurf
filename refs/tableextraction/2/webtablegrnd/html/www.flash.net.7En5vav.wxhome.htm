Welcome to the Weather Webring Home. This webring is dedicated to weather related sites, especially storm spotting, weather stations, and weather information sites.. If you have a weather related site and would like to have your site a part of the ring, use the form below to submit your site.

If you are already a member of the ring and would like to edit your information, use the Edit Information section at the bottom of this page.

To submit a site, simply use the form below: Please note that I, or another member of the ring, will check your site for appropriateness in the ring. Get the HTML code HERE. ALSO, your pre-made, customized HTML fragment will be E-mailed to you ! Just copy and paste to your weather page. Please remember, your site will not be added until the HTML code is present on your site. PLEASE be sure the RING URL you supplied points to the page with the HTML code. It makes the ring much easier to navigate. Thank you !

FRAMES users... This webring has an automatic page checker. In order for the checker to see your fragment, please put your html code on your FRAMESET page as well , OR, direct the webring to the actual page with the HTML code !! This way, the ring checker will keep you in the ring.
