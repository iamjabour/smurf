Results Page

The Results window gives you an overview of the applications you started, their status (status column) and their output files. From here you can view and print your results. You can save output files under different names, or delete them. You can also kill a running job in this window.

To view a result file click onto the output file name.

The Refresh button updates the contents of your Results window. Use the "Remove" function to clean up this window.
