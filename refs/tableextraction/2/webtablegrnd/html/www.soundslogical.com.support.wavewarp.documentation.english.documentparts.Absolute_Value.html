Dynamic Range Controllers:

Absolute Value



Functional Description
The output control signal (y) is equal to the absolute value of the input audio signal (x):

Math function
y = abs(x)
Algorithm
Floating-point math function
Signal Implementations
Audio signals Control signals Description
Single input mono single output Control output equals absolute value of mono audio input
Single input stereo single output Control output equals average of the absolute values of the left and right audio input channels
