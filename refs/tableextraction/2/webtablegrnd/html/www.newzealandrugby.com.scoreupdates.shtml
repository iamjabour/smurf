Live Scoring Updates



Throughout the rugby season New Zealand Rugby.com in association with other Web Rugby sites will provide scoring updates of selected games.



Games that will be covered by Live Updates will be advertised on the front page of New Zealand Rugby.com To access the score updates you will need a javascript enabled browser.



Games typically covered score updates include up to six Super 12 games each weekend, up to four NPC games, Tri-Nations, All Black tests and selected other games throughout the season.



Further information regarding Live Scoring Updates can be obtained from Web Rugby
